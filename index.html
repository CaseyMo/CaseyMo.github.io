基于Windows操作系统利用Hexo框架+GitHub Pages搭建个人博客

一、基本概念
Hexo是一个快速、简洁且高效的博客框架，使用 Markdown（或其他渲染引擎）解析文章，在几秒内即可利用靓丽的主题生成静态网页。更多介绍详见https://hexo.io/。
GitHub Pages是一个静态网站托管服务，能够从建立在GitHub上的仓库中直接存取HTML、CSS和JavaScript文件。你可以通过构建过程来运行这些文件和发布网站。更多介绍详见https://docs.github.com/cn/pages/getting-started-with-github-pages/about-github-pages。

二、Hexo安装
安装Hexo前需要确保已安装了以下应用程序：
Node.js–是一个开源和跨平台的JavaScript运行环境,在浏览器之外运行V8 JavaScript引擎（Google Chrome 的内核）,更多介绍详见http://nodejs.cn/learn/introduction-to-nodejs。
Git–是一个免费、开源的分布式版本控制系统，更多介绍详见https://git-scm.com/。

关于安装Node.Js和Git的方法详见Hexo的使用文档https://hexo.io/docs/。
